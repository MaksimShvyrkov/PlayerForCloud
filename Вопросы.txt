1. Опишите самую интересную задачу в программировании, которую вам приходилось решать?

Настривал парсинг сайтов перевозчиков на базе приложения PIX, полученные данные
интегрировал в приложение "Первая форма" посредстова публикаций (API),
на SQL написал отчёт о работе автоматизации и при помощи API Telgram отправлял данные в чат.

Автоматизировал процесс заведение документации для крупного магазина косметики посредством JS, SQL, LUA.
JS: работа с DOM элементами, привязка к событиям, вызов публикаций при помощи fetch, работа с async.
LUA: Создание API для взаимодействия с приложением "Первая Форма" и работа с потоками сообщений (настройка очередей)
SQL: Создание хранимых процедур и функций


2. Расскажите о своем самом большом факапе? Что вы предприняли для решения проблемы?

Не учёл некоторые переменные при рефакторинге хранимой процедуры в связи с чем у пользователей из
двух отделов ожидаемая зарплата за месяц стала больше в 4 раза.

Решение:
Первоначально - оперативно был восстановлен бэкап, далее исправление ХП с учётом всех переменных.
В дальнейшем стал использовать больше тестовых данных при проверке кода.
